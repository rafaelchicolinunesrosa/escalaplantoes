Como usar os Templates:
  • Template de Plantão: Salva configuração básica (Viatura, Agentes, 36) sem O.S. Use para ter uma base pré-preenchida ao selecionar o plantão.
  • Escala do Dia: Salva tudo (incluindo O.S.) para uma data específica. Ideal para preparar o mês inteiro no início, considerando folgas e férias.
  • Ver Escalas Salvas: Clique em "Ver Escalas Salvas" para listar todas as datas com escalas prontas e carregar rapidamente.
